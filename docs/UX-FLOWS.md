## 6. Mobile Considerations

### Touch-First Adjustments
- Larger tap targets (44px minimum)
- Bottom navigation for thumb reach
- Swipe gestures for common actions
- Simplified mission cards

### Responsive Breakpoints
- **Mobile**: < 640px (stack everything)

## 7. Accessibility Flows

### Keyboard Navigation
- Tab through all interactive elements
- Enter to activate
- Escape to close modals
- Arrow keys in lists

### Screen Reader
- Semantic HTML throughout
- ARIA labels on icons
- Live regions for XP updates
- Skip links for navigation

### Reduced Motion
- Respect `prefers-reduced-motion`
- Disable boot animations
- Static XP bar (no pulse)

## 8. Error States

### Common Error Flows
1. **Submission Failed**: Retry with saved draft
2. **Network Offline**: Cache actions, sync later
