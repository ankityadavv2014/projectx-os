## [0.1.0] - 2026-01-30

### Added
- Initial project scaffold with Next.js 16, TypeScript, Tailwind CSS
- Landing page with Hero, Manifesto, Trailer, EnterOS sections
- Cyberpunk OS desktop experience
- XP progression system with Zustand store
- AI Orb component (placeholder)
- Missions app component
- Core UI components (Button, Card, GlowText, ParticleField)
- Core documentation (PRD, ARCHITECTURE, UX_SPEC, SECURITY, NORTH-STAR, BACKLOG)
- 10 agent prompts for agentic organization
- Agent factory CLI
- Security headers in Next.js config
- CI/CD pipeline with GitHub Actions
- Pilot integration for autonomous development
- Entity types for global platform (400+ lines)

### Technical
- Zustand for state management (xp-store, os-store, orb-store)
- Framer Motion for animations
- Custom CSS variables for brand colors
- TypeScript strict mode
